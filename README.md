# SIMConnect-Calendar-to-ICS
Web-based tool for converting SIM timetable to ics file. No downloads or installations.

Page URL: [https://looi-wh.github.io/SIMConnect-Calendar-to-ICS](https://looi-wh.github.io/SIMConnect-Calendar-to-ICS)

### Steps:
1. Browse to the following in `https://simconnect.simge.edu.sg`
```
SIM Connect > My Apps > Enrollment > My Timetable (List View:Class)
```
2. `Ctrl` + `a` to select all text in the page
3. `Ctrl` + `c` to copy all text in the page
4. Browse to this [tool 1](https://looi-wh.github.io/SIMConnect-Calendar-to-ICS/#convert)
5. Paste into the field and click `Download ICS`
6. ICS file will be valid for any applications such as Google Calendar and iOS Calendar
